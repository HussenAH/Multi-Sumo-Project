Multi-Sumo Project

Introduction

Multi-Sumo application enables control over unlimited number of Parrot Jumping Sumo and manages operation of them, taking into consideration the various features and functionality of the Jumping Sumo.

The application’s code is an open source, provides to user easy-to-use API to control over the Sumo, and easy maintenance and management.

Here you will find a guide of how to use the application, and start the system. You can also find a video that shows and describes the functionality and usage of the App. It’s highly recommended.

More information

The project directory contains two applications:
▪ Sumo-Master Application

The Sumo-Master application is intended to control and mange operation of the entire system, it also manages all of the clients connection and sends string commands to each of them. This application includes easy to use UI, which must be downloaded and installed on the Master side.

▪ Sumo-Client Application

The Sumo-Client application manages the connection between the clients and the Master by sending connection requests to the Master. It also responsible for connecting with the robot, by converting the received string commands (from the Master) to commands that can control over the robot, making him move, jump and rotate etc.
