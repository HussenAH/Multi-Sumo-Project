Multi-Sumo Project

introduction

The project deals with the management, control, and operation of an unlimited number of Parrot Jumping Sumo
, by one computer, master, and a number of computers ODROID robots.
The code is open source which exercised all the functionality needed to maintain,
modify, and develop other needs related code.
Here you will find a guide to how to connect, play, and play system, step by step,
as well as the project file attached video documenting all stages step by step in real time, it is highly recommended to watch simultaneously.

More informations

The project file There are two applications:
▪ The Sumo-client app:
Is responsible for managing all of the system, the client-sea connection, and sending commands to each of them.
This application includes easy to use UI, and "planted" in a computer which is the master
▪ The Sumo-master app:
On the one hand, this app is responsible for managing the connection in front of the master, on the other hand, runs the front of the sumo connection connected to the specific client which is planted in it.
